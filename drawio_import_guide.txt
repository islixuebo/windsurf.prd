# Draw.io 导入登录流程图指南

## 🎯 快速导入方法

### 方法1: 直接URL导入
在Draw.io编辑器中：
1. 点击 "文件" → "导入"
2. 选择 "XML"
3. 复制以下XML内容并粘贴

或者直接访问：
```
http://127.0.0.1:6956/#Uhttps://raw.githubusercontent.com/jgraph/drawio/master/src/main/webapp/index.html
```

### 方法2: 使用文件导入
1. 在Draw.io中点击 "文件" → "导入"
2. 选择 "从设备导入"
3. 选择文件：`login_flow_drawio.xml`

### 方法3: 手动绘制步骤

#### 步骤1: 创建参与者
在左侧面板选择：
- **UML** → **参与者** (Actor) → 创建"用户"
- **UML** → **边界** → 创建"前端"
- **UML** → **边界** → 创建"后端API"  
- **UML** → **边界** → 创建"数据库"

#### 步骤2: 绘制消息流
使用连接线工具：
1. 从用户拖动到前端 → "输入用户名和密码"
2. 前端自调用 → "前端表单验证"
3. 前端到后端 → "POST /api/login"
4. 后端自调用 → "验证用户名密码格式"
5. 后端到数据库 → "SELECT * FROM users"
6. 数据库返回后端 → "返回用户信息"
7. 后端自调用 → "密码加密比对"
8. 后端自调用 → "生成JWT Token"
9. 后端返回前端 → "返回Token"
10. 前端自调用 → "存储Token"
11. 前端到用户 → "跳转首页"

#### 步骤3: 添加错误处理
- 后端到前端（红色）→ "密码错误"
- 前端到用户（红色）→ "显示错误信息"

#### 步骤4: 添加注释
使用注释工具添加：
- 安全措施说明
- JWT Token内容
- 技术栈信息

## 🎨 样式设置

### 颜色方案
- 用户：蓝色 (#dae8fc)
- 前端：黄色 (#fff2cc)
- 后端API：紫色 (#e1d5e7)
- 数据库：绿色 (#d5e8d4)
- 成功流程：绿色箭头
- 错误流程：红色箭头

### 连接线样式
- 实线：同步调用
- 虚线：异步返回
- 自调用：循环处理

## 📱 实时绘制效果

Draw.io支持以下实时效果：
1. **拖拽实时显示** - 拖动图形时实时更新位置
2. **连接实时显示** - 绘制连接线时即时显示
3. **文本实时编辑** - 输入文本时立即更新
4. **样式实时预览** - 修改颜色、字体等实时生效
5. **自动对齐** - 智能对齐网格和连接点
