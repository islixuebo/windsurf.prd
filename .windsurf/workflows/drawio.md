# Draw.io 图表编辑器工作流

此工作流用于创建和编辑各种类型的图表，包括流程图、时序图、架构图等。

## 使用方法

### 方式一：直接请求创建图表
```
请用Draw.io创建一个[图表类型]图，内容是[具体描述]
```

**示例：**
- `请用Draw.io创建一个OAuth2认证流程的时序图`
- `请用Draw.io创建一个微服务架构图`
- `请用Draw.io创建一个用户注册流程图`

### 方式二：打开编辑器手动创建
```
请打开Draw.io编辑器
```

## 服务配置

**Draw.io Editor服务信息：**
- HTTP端口：8910
- WebSocket端口：3333（默认）
- 访问地址：http://localhost:8910
- 状态：自动启动

## 支持的图表类型

### 1. **流程图**
- 业务流程
- 算法流程
- 工作流程

### 2. **时序图**
- 系统交互
- API调用流程
- 消息传递

### 3. **架构图**
- 系统架构
- 网络拓扑
- 微服务架构

### 4. **UML图**
- 类图
- 用例图
- 状态图

### 5. **其他图表**
- 思维导图
- 组织结构图
- 网络图

## 执行步骤

1. **启动Draw.io服务**
   - 自动启动在端口8765
   - 提供Web编辑器界面

2. **创建图表**
   - 根据描述自动生成图表
   - 或打开编辑器手动创建

3. **编辑和优化**
   - 可视化拖拽编辑
   - 调整样式和布局

4. **保存和导出**
   - 保存到项目文件
   - 导出PNG/SVG/PDF格式

### 基本功能

- ✅ 丰富的图形库

- ✅ PNG图片
- ✅ SVG矢量图

- ✅ XML源码

## 使用技巧

### 快捷键
- `Ctrl+C/V` - 复制/粘贴
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做
- `Delete` - 删除选中

### 布局建议
- 使用网格对齐
- 保持一致的间距
- 使用颜色区分组件
- 添加清晰的标签

### 图表规范
- 从左到右或从上到下布局
- 使用标准图形符号
- 保持简洁明了
- 添加必要的说明

## 注意事项

- 服务会自动启动，无需手动操作
- 编辑器支持中文输入
- 可以实时保存和预览
- 导出的文件会保存到项目目录

## 故障排除

如果编辑器无法打开：
1. 检查端口8910是否被占用
2. 重启Windsurf重新加载配置
3. 确认网络连接正常
