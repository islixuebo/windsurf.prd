# 电商架构图绘制指南

## 在Draw.io中直接绘制步骤：

### 1. 打开Draw.io编辑器
访问：http://127.0.0.1:6956

### 2. 选择模板
- 点击 "创建新图表"
- 选择 "流程图" 或 "架构图" 模板

### 3. 绘制架构层次

#### 📱 前端层 (蓝色)
```
┌─────────────────────────────────────────────────────────────┐
│                    前端层 (Frontend)                        │
├─────────────────┬─────────────────┬─────────────────────────┤
│   Web应用       │   移动应用      │      小程序             │
│   (React/Vue)   │  (iOS/Android) │  (微信/支付宝)          │
└─────────────────┴─────────────────┴─────────────────────────┘
```

#### 🚪 API网关层 (绿色)
```
┌─────────────────────────────────────────────────────────────┐
│                   API网关层 (Gateway)                        │
├──────────┬──────────┬──────────┬───────────────────────────┤
│ 负载均衡  │ 路由转发  │ 认证授权  │      限流熔断              │
└──────────┴──────────┴──────────┴───────────────────────────┘
```

#### ⚙️ 微服务层 (红色)
```
┌─────────────────────────────────────────────────────────────┐
│                  微服务层 (Microservices)                    │
├─────────┬─────────┬─────────┬─────────┬─────────┬───────────┤
│用户服务 │商品服务 │订单服务 │购物车   │支付服务 │物流服务   │
│•注册登录│•商品管理│•订单创建│•购物车  │•支付集成│•配送管理  │
│•个人信息│•分类管理│•订单支付│•商品操作│•退款处理│•物流跟踪  │
│•地址管理│•库存管理│•状态管理│•数量管理│•银行卡 │•仓库管理  │
└─────────┴─────────┴─────────┴─────────┴─────────┴───────────┘
```

#### 🗄️ 数据存储层 (紫色)
```
┌─────────────────────────────────────────────────────────────┐
│                   数据存储层 (Storage)                       │
├──────────────┬──────────────┬──────────────┬────────────────┤
│ 关系型数据库 │  NoSQL数据库  │   消息队列   │   第三方服务    │
│ •MySQL      │ •Redis       │ •RabbitMQ    │ •短信服务      │
│ •PostgreSQL │ •MongoDB     │ •Kafka       │ •邮件服务      │
│ •主从复制   │ •Elasticsearch│ •事件驱动    │ •地图服务      │
│ •读写分离   │ •数据分片    │ •解耦服务    │ •CDN服务       │
└──────────────┴──────────────┴──────────────┴────────────────┘
```

### 4. 连接关系
- 前端层 → API网关层 (所有连接)
- API网关层 → 微服务层 (分发到各个服务)
- 微服务层 → 数据存储层 (根据服务类型连接)

### 5. 使用Draw.io工具
- **矩形工具**: 绘制各个层和服务
- **文本工具**: 添加服务名称和功能
- **连接线工具**: 绘制数据流向
- **颜色填充**: 区分不同层次
- **图标**: 添加相关图标增强可视化

### 6. 推荐颜色方案
- 前端层: #4285f4 (蓝色)
- 网关层: #34a853 (绿色)  
- 微服务层: #ea4335 (红色)
- 数据层: #9c27b0 (紫色)

### 7. 布局建议
- 每层高度: 100-150px
- 服务宽度: 120-150px
- 间距: 20px
- 连接线: 直线或折线

现在你可以在Draw.io编辑器中按照这个指南直接绘制完整的电商架构图！
